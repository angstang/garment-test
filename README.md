一个专业的服装行业标准知识测试平台，针对儿童产品和校服安全标准的在线评估系统。

## 📋 项目概述

**新益服装产品技术理论测试系统**是为服装企业内部员工（设计、工艺、版师、品质等部门）设计的知识评估工具，重点关注儿童服装和校服的安全标准，包括化学安全、物理安全和标签要求。

### 主要特点

- 🎯 **70道精选题库** - 基于官方校服标准清单，仅包含经过验证的标准
- 📱 **响应式设计** - 完美适配PC、平板和手机，支持在微信中打开
- 📊 **详细报告** - 支持JSON、PDF和图片三种格式导出
- 🔄 **动态随机抽取** - 每次测试从115道题库中随机抽取30道题
- 💯 **实时反馈** - 即时显示答案对错和详细解释
- 📈 **分类统计** - 按知识类别统计成绩，精准定位弱项

### 关键知识点

**GB 31701 物理安全指标：**
- 颈部绳索长度：≤22cm
- 腰部绳索长度：≤56cm
- 小部件拉力强度：≥90N
- 孔洞直径限制：≤5mm禁止
- 拉链齿间距：≥3mm
- 领口最小尺寸：≥280mm

**GB 31701 化学安全指标：**
- 甲醛含量：≤75mg/kg
- 可分解芳香胺：≤30mg/kg
- pH值范围：4.0-7.5
- 禁用染料：23种已知致敏染料

**校服关键指标（GB/T 23328）：**
- 色牢度（耐皂洗）：≥3级
- 接缝强力：≥面料强力75%
- 缩水率：经向≤3%，纬向≤4%
- 防起球性：≥3级
- 面料克重：120-200 g/m²

**校服面料知识要点：**
- 纤维成分：纯棉（透气舒适）、涤纶（耐久性强）、棉涤混纺（综合性能）
- 面料密度：影响强度和透气性的关键指标
- 色牢度检测：GB/T 3917.3耐皂洗、GB/T 4802.2皮肤摩擦、GB/T 31902汗渍
- 缩水率检测：GB/T 7573规定水洗后的尺寸变化限值
- 抗磨损性：GB/T 4802.1防起球性，需≥3级

**检测方法标准覆盖：**
- GB/T 5453：织物拉伸强力测定
- GB/T 3917.3：色牢度试验（耐皂洗）
- GB/T 7573：水洗尺寸变化测定
- GB/T 4802系列：耐磨、起球、抗菌性能
- GB/T 5326：撕裂强力测定
- GB/T 31902：汗渍色牢度测定

## 📖 使用流程

1. **进入应用** - 打开HTML文件或在线链接
2. **输入信息** - 填写姓名和部门（设计部/工艺部/版师部/品质部/管理层/其他）
3. **开始测试** - 系统随机抽取30道题，按顺序答题
4. **即时反馈** - 每题提交后显示对错和详细解释
5. **查看成绩** - 测试完成后显示总分、分类成绩和评价
6. **导出报告** - 支持三种格式导出：
   - 📥 **JSON格式** - 保存原始数据
   - 📄 **PDF格式** - 专业报告文档
   - 🖼️ **PNG图片** - 截图分享
