# 女娲智库 (Nuwa Skill)

> 借鉴中国古代女娲补天的理念——以包容、全局的视角重构问题，汇集当代顶级思想家的智慧。

---

## 设计理念

**女娲 = 多视角智库**

当你面对复杂问题时，单一视角往往不足。女娲智库通过汇集13位顶级思想家的视角（来自科技、商业、哲学、教育等领域）+ 1位主题导师的深度分析，帮助你从全局视角重新理解问题。

核心三层：
1. **问题诊断** — 用7维度框架拆解你的问题
2. **多视角求解** — 从13个视角逐个分析建议
3. **智慧整合** — 综合所有视角，得出最优决策路径

## 工作流程

### Phase 0: 理解你的问题

```
用户输入: "问题描述"
↓
系统分解为:
  1. 问题类型 (创业/产品/人生/技术/等)
  2. 当前困境 (具体的卡点)
  3. 期望目标 (想达到什么)
  4. 约束条件 (有哪些限制)
```

### Phase 1: 多视角分析

```
for each 视角 in [13位思想家 + 1位导师]:
  调用对应的 perspective skill
  获得该视角的分析建议
  汇总关键洞察
```

每个视角包含：
- **核心建议** — 该视角下最重要的3个建议
- **思维框架** — 这个人会如何看待你的问题
- **案例参考** — 类似问题的真实案例
- **潜在风险** — 这个视角容易忽视什么

### Phase 2: 智慧整合

```
对比14个视角的建议
│
├─ 共识点 (多数视角都提到的)
├─ 分歧点 (不同视角有矛盾的)
├─ 补充点 (某个特定视角的独特贡献)
│
↓
生成综合方案:
  1. 优先级最高的行动 (立即做)
  2. 中期规划 (1-3个月)
  3. 长期愿景 (1年以上)
```

### Phase 3: 个性化指导

```
基于你的行业/阶段，推荐：
  - 重点关注的2-3个视角
  - 这个问题的核心杠杆点
  - 常见的坑和陷阱
  - 下一步具体行动
```

## 使用场景

### 创业者
> "我想创业，不知道从哪里开始"

女娲会综合：
- Paul Graham 的创业本质 (赚钱 = 帮人解决问题)
- 张一鸣 的产品快速迭代法
- Karpathy 的技术选型和工程效率
- Naval 的商业杠杆论
- 张雪峰 的现实主义视角

结合给出具体的：创业方向选择 → MVP设计 → 融资节奏 → 团队搭建

### 产品经理
> "产品增长遇到瓶颈"

女娲会综合：
- Jobs 的用户体验美学
- Graham 的效率优化
- MrBeast 的增长黑客
- 马斯克 的第一性原理
- 芒格 的跨学科思维

给出：用户研究 → 价值主张重塑 → 增长杠杆 → 竞争护城河

### 个人发展
> "不知道职业方向，感到迷茫"

女娲会综合：
- Naval 的财富和自由定义
- Fee yman 的学习方法
- 张雪峰 的职业洞察
- Jobs 的匠人精神
- Graham 的效率和专注

给出：自我认知 → 能力规划 → 机会选择 → 人生杠杆

## 常见问题

**Q: 14个视角的建议会不会相互矛盾？**  
A: 会，这正是价值所在。矛盾说明需要权衡。女娲会明确指出分歧点，帮你理解不同视角的假设和适用场景。

**Q: 用女娲分析需要多长时间？**  
A: 取决于问题复杂度。简单问题 5-10 分钟，复杂问题可能需要 30 分钟到 1 小时。

**Q: 可以自定义新视角吗？**  
A: 可以。如果你想加入某个思想家或行业专家的视角，见 [references/skill-template.md](references/skill-template.md)。

## 设计灵感

古代女娲补天的故事：面对宇宙的混乱和缺陷，女娲不是单一地「强行修补」，而是汇集多种方式（五色石、芦苇灰等）来修复。

现代应用：面对复杂问题时，不依赖单一思想家或方法论，而是汇集多个顶级思维框架，找到最优解。
