# Perspective Skill 生成模板

> 如何创建一个新的 perspective skill，加入女娲智库？

---

## 快速开始

### 步骤 1：准备工作

选定你的目标人物或主题：
- 了解他/她的核心思想
- 收集主要著作、论文、演讲
- 浏览相关案例和评价

### 步骤 2：提炼内容

参考 [extraction-framework.md](extraction-framework.md) 中的**5层提炼法**，准备：
- 人物DNA（核心信念、思维工具、盲点、成功基因）
- 核心框架（3-5个思维模型）
- 具体建议（针对5个常见问题类型）
- 案例库（5-10个案例）
- 风险和补充（适用范围、盲点、配套视角）

### 步骤 3：使用本模板

复制下面的文件结构和 SKILL.md 内容，根据你的人物/主题填充。

---

## 目录结构

```
[人物名称]-perspective/  (或 [主题]-mentor/)
├── SKILL.md              # 核心内容（用户看这个）
├── README.md             # 简介（可选）
├── LICENSE               # 许可证（可选）
├── references/
│   ├── bio.md           # 人物简历和背景
│   ├── frameworks.md    # 核心框架详解
│   ├── cases.md         # 案例库
│   └── sources.md       # 参考文献
└── examples/            # 应用示例（可选）
    ├── case-study-1.md
    └── case-study-2.md
```

## 常见问题

**Q: 我可以为我喜欢的任何人创建 perspective 吗？**  
A: 可以的，但建议选择有明确思想框架、公开著作或言论的人物，这样更容易提炼出可用的建议。

**Q: 我的人物 SKILL.md 需要多详细？**  
A: 最少：人物DNA + 2-3个框架 + 3个问题类型的建议。最佳：上面模板中的所有部分。

**Q: 多久需要更新一次？**  
A: 建议每季度审视一次，看是否有新的著作、演讲或实践数据值得补充。

**Q: 我可以引用受版权保护的内容吗？**  
A: 可以，但需要正确标注来源和作者。确保符合合理使用原则。

**Q: 如何测试我的 skill 质量？**  
A: 邀请1-2个朋友在实际问题上试用，看看建议是否有帮助。收集反馈后改进。

## 速查表：各部分的最小内容

| 部分 | 最小要求 | 最佳要求 |
