## 🆘 常见问题

### Q1: 自动跳转时间太短了？
```
A: 可以修改 submitAnswer() 中的 3000 (毫秒)：
   setTimeout(() => {
       // 改为 5000 表示5秒
   }, 3000);
```

### Q2: 分享在某些浏览器不工作？
```
A: Web Share API 在某些浏览器不支持，此时会自动
   使用备选方案（复制到剪贴板）。用户体验不受影响。
```

### Q3: 修改答案限制可以解除吗？
```
A: 可以在 showFeedback() 中修改 appState.canReedit 的逻辑，
   但不建议，因为这样会降低考试的严谨性。
```

### Q4: 如何禁用自动跳转？
```
A: 注释掉 submitAnswer() 中的 setTimeout 代码即可：
   // appState.autoNextTimer = setTimeout(() => {
   //     ...
   // }, 3000);
```
