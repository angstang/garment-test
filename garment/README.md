# 新益服装知识测试系统 v4.1

> 🎯 多岗位智能答题系统 + 快速答题模式 + 错题详解 + 飞书自动同步

## 📱 系统概览

一个专为服装行业设计的在线考试系统。采用**快速答题 + 最后总结错题**的新模式。

### 核心功能
- ✅ **快速答题流程** - 提交答案后直接跳转到下一题，无即时反馈
- ✅ **错题详细分析** - 成绩报告中显示所有错题的详细解释和正确答案
- ✅ **多岗位题库** - 设计师、工艺师、版师等岗位专属题目
- ✅ **时间限制** - 15分钟/30题，每题30秒倒计时
- ✅ **用户信息收集** - 自动收集答题人员详细信息
- ✅ **飞书自动同步** - 成绩一键导入飞书多维表格
- ✅ **分享功能** - 支持分享到微信、QQ或复制到剪贴板
- ✅ **本地数据备份** - 所有数据自动保存到本地存储

## 🎯 新的答题流程 (v4.1新增)

### 快速答题模式

```
答题过程：
选择答案 → 点"提交答案" → ⚡ 直接跳转到下一题（无反馈）
   ↓
重复...
   ↓
```

### 成绩报告包含

✅ **总体成绩** - 分数、百分比、等级评价  
✅ **分类成绩统计** - 各知识点答对情况  
✅ **❌ 错题分析详解** - 每道错题显示：
   - 题目内容

   - 正确答案（绿色标注）
   - 题目详细解释
   - 考查的具体标准

### 为什么采用快速答题模式？

| 特点 | 优势 |
|------|------|
| ⚡ 快速跳转 | 答题更专注，不被反馈打断 |
| 📊 最后总结 | 完整看到所有错题，便于统一学习 |
| 🎓 学习效率 | 先完整作答，再深入反思 |

```javascript

    totalTime: 20 * 60,      // 改为 20 分钟

};
```

### 修改出题数

编辑 `config.js`：
```javascript

    questionsPerTest: 50     // 改为 50 道题
};
```


## ⚠️ 常见问题

### Q: 提交答案后为什么没有反馈？

**A:** 这是 v4.1 的新设计！采用"快速答题 + 最后总结"模式：
- ✓ 提交后直接跳转到下一题（保持答题连贯性）
- ✓ 所有错题详解在最后的成绩报告中统一显示
- ✓ 完成全部答题后可详细查看每题的解释

### Q: 能改答案吗？

**A:** 不能。因为采用快速答题模式，提交即跳转：
- 提交答案是确定选择
- 如需改答案，可点"⬅️ 上一题"返回前面的题目重新作答
- 改好后再点"提交答案"会覆盖之前的答案

### Q: 错题在哪里看？

**A:** 完成所有答题后，在**成绩报告页面**向下滚动：
- 📊 会看到"❌ 错题分析"部分
- 每道错题显示：题目、你的答案（红）、正确答案（绿）、解释、标准
- 支持导出PDF或图片时也包含错题分析

### Q: 飞书没有收到数据？

**A:** 检查以下几点：
1. F12 → Console 是否有错误信息
2. `config.js` 中 `enabled: true` 是否已设置
3. Webhook URL 是否正确（已自动配置）
4. 数据一定已保存到本地（即使飞书同步失败）

### Q: 时间到了会怎样？

**A:** 系统自动：
1. 未答题的题自动选第一个选项
2. 直接跳转到成绩页面
3. 显示 "⏱️ 时间到！系统已自动交卷"
4. 无法继续答题

### Q: 如何查看本地数据？

**A:** 打开浏览器 F12：
```
F12 → Application 标签
→ Local Storage
→ 选择当前网址
→ 找到 testRecords 键
```

或在 Console 执行：
```javascript
JSON.stringify(JSON.parse(localStorage.getItem('testRecords')), null, 2)
```

### Q: 如何分享成绩？

**A:** 完成答题后在成绩页面点"📱 分享"按钮：
- 🔗 **Web Share API**（手机优先）- 系统分享菜单
- 📱 **微信** - 提示在微信内打开
- 🐧 **QQ** - 生成QQ分享链接
- 📋 **复制** - 复制成绩文本到剪贴板
