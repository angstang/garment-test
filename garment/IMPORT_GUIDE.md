# 女装甲级测试 - 525题导入指南

## 📋 概述

`import_525_questions.py` 是一个自动化脚本，用于将女装甲级测试的525道题目从markdown格式导入到题库JSON数据库。

## 🚀 快速开始

### 步骤 1: 准备Markdown文件

将包含525道题目的markdown文件放在以下位置之一：
- `女装甲级测试.md`
- `女装甲級測試.md`
- `garment_exam_525.md`
- `exam_525.md`

### 步骤 2: 运行导入脚本

```bash
python3 import_525_questions.py
```

或者指定文件路径：

```bash
python3 import_525_questions.py /path/to/exam_file.md
```

### 步骤 3: 验证导入结果

脚本会显示：
- ✅ 成功解析的题目数量
- 📊 按工作项目分类的统计
- 📈 导入后的题库总体状态

## 📝 Markdown文件格式要求

### 总体结构

```markdown
# 女装甲级测试

## 工作项目01: 业界知识 (78题)

1. (易) 题目内容这里
A. 选项A
B. 选项B
C. 选项C
D. 选项D
答: A
解: 详细解释文本

2. (中) 下一道题目...
...

## 工作项目02: 工具使用 (60题)

1. (易) 题目...
...
```

### 详细说明

#### 工作项目段落
- 格式: `## 工作项目XX: 名称 (题数)`
- 例如: `## 工作项目05: 素材 (174题)`

#### 单道题目
- **题号**: `数字. (权重/难度) 题目文本`
  - 例如: `1. (易) 天然纤维包括哪些?`
  - 权重可选，支持: 易、中、难等任何文本
  
- **选项**: 按顺序排列，每行一个
  ```
  A. 选项内容
  B. 选项内容
  C. 选项内容
  D. 选项内容
  ```
  
- **答案**: 标记正确答案
  ```
  答: A
  ```
  或
  ```
  答: 1
  ```
  支持字母(A-D)或数字(1-4)格式
  
- **解释**: 详细说明为什么这个答案正确
  ```
  解: 详细解释文本，可以多行
  ```

## ⚙️ 分配方案说明

脚本使用 **Scheme B (混合分配)** 方案：

✓ 所有525道题目将被分配到：
- 设计师题库
- 工艺师题库
- 版师题库

每个部门都可以访问完整的525道题目，实现全岗位知识覆盖。

## 📊 解析过程

脚本执行的步骤：

1. **读取markdown文件** - 加载题目数据
2. **分割工作项目** - 按工作项目(01-07)分类
3. **解析题目** - 提取题号、内容、选项、答案、解释
4. **验证数据** - 确保选项完整、答案有效
5. **分配ID** - 为每道题分配唯一ID
6. **分布到部门** - 添加到所有部门的题库数组
7. **保存数据库** - 更新questions_multi_dept.json

## ⚠️ 常见问题

### Q: 导入后为什么网页打不开题目？

A: 请检查：
1. 题库JSON语法是否正确（JSON验证工具）
2. 数据库文件是否成功更新（检查修改时间）
3. 浏览器缓存（F12 清除缓存后重新刷新）

### Q: 题目数量不对怎么办？

A: 脚本会显示警告信息如果题目数量不匹配。请检查markdown文件格式：
- 每道题是否有完整的A/B/C/D四个选项
- 答案标记是否正确（答: A 或 答: 1）
- 是否有多余的空白行干扰解析

### Q: 能否只导入某些工作项目？

A: 可以修改markdown文件，只保留需要的工作项目段落。脚本会自动处理任何数量的工作项目。

### Q: 已有的题目会被覆盖吗？

A: 不会。脚本会自动计算最大ID，新题目的ID会接续现有题目。但建议在导入前备份原数据库。
