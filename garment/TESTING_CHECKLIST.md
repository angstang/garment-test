## ✅ 功能测试

### 1️⃣ 自动跳转功能 (自动下一题)
```
□ 提交答案后，反馈页显示 "⏱️ 3秒后自动下一题" 提示
□ 不操作的情况下，3秒后自动跳转到下一题
□ 进度条自动更新
```

### 2️⃣ 修改答案功能
```
□ 反馈显示时可点击其他选项修改答案
□ 点击修改后，反馈消失，显示新的选择
□ 自动跳转被取消（等待用户操作）
□ 可再次点击"提交答案"看新的反馈
□ 修改后的答案被记录到成绩中
```

### 3️⃣ 上一题功能
```
□ 第1题时，"⬅️ 上一题"按钮不显示
□ 第2题及以后，"⬅️ 上一题"按钮显示
□ 点击后返回上一题
□ 之前的答案被保留
```

### 4️⃣ 超时自动跳转
```
□ 每题有30秒计时
□ 计时条显示在题目上方（橙色→红色）
□ 30秒到自动提交答案
□ 若未选答案，自动选择第一个选项
```

### 5️⃣ 导出功能
```
□ 不显示"📥 下载JSON"按钮
□ 显示"📄 导出PDF"按钮 ✓
□ 显示"🖼️ 保存图片"按钮 ✓
□ 点击"📄 导出PDF"生成PDF报告
□ 点击"🖼️ 保存图片"生成PNG图片
```

### 6️⃣ 分享功能
```
□ 成绩页显示"📱 分享"按钮
□ 点击弹出分享选项
□ 选项包括:
  □ 微信分享（提示在微信内打开）
  □ QQ分享（生成QQ分享链接）
  □ 复制到剪贴板（可粘贴到任何平台）
□ 分享文本包含成绩、部门、答题统计
```

## 📊 预期行为示例

### 场景1：正常流程
```
1. 第1题：选A → 提交 → 显示"正确"反馈 → 3秒后自动下一题
2. 第2题：选B → 提交 → 显示"错误"反馈及正确答案 → 3秒后自动下一题
3. ...重复直到最后一题
4. 自动进入成绩页
```

### 场景2：修改答案流程
```
1. 选A → 提交 → 显示"错误"反馈
2. 在反馈显示期间点击B → 反馈消失，显示B被选中
3. 点击"提交答案" → 显示新反馈（B对应的反馈）
4. 不操作 → 3秒后自动下一题
```

### 场景3：翻题流程
```
1. 正在第5题
2. 点击"⬅️ 上一题" → 返回第4题，答案被保留
3. 可修改第4题答案并重新提交
4. 点击"➡️ 下一题" → 返回第5题
```

## 📞 问题排查

如果发现问题，检查：

1. **按钮不显示**：检查浏览器缓存，强刷（Ctrl+Shift+R）
2. **计时不准确**：检查系统时间是否正确
3. **分享不工作**：某些浏览器可能不支持 Web Share API
4. **样式混乱**：检查 CSS 加载是否完整
5. **控制台错误**：检查是否有拼写错误或引用错误
