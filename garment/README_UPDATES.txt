===============================================
  服装答题系统更新完成 - v4.1
===============================================

✅ 所有6个需求功能已实现：

1. ⏱️  超时自动跳转到下一题
   - 30秒题目计时，超时自动提交
   
2. 🚀 作答后自动跳转到下一题
   - 显示3秒反馈，然后自动跳转
   - 用户可在3秒内修改答案取消自动跳转
   
3. ⬅️ 可往后翻一题（上一题功能）
   - 新增"⬅️ 上一题"按钮
   - 自动显示/隐藏，取决于题号
   
4. 🔄 也可改一次（修改答案功能）
   - 看到反馈后可点击其他选项修改
   - 修改后可再次提交看新反馈
   - 限制修改一次
   
5. 📊 导出只保留PDF和图片
   - 移除JSON下载按钮
   - 保留PDF导出和图片保存
   
6. 📱 分享到微信或其他平台
   - 新增分享按钮
   - 支持：微信、QQ、复制到剪贴板

主要改动概览：
===============================================

题目页面按钮组：
  原: [提交答案]
  新: [⬅️ 上一题] [提交答案] [➡️ 下一题]

成绩页面按钮组：
  原: [📥 下载JSON] [📄 导出PDF] [🖼️ 保存图片] [🔄 重新测试]
  新: [📄 导出PDF] [🖼️ 保存图片] [📱 分享] [🔄 重新测试]

核心流程改变：
  - 提交答案 → 显示反馈(3秒) → 自动下一题
  - 用户在3秒内可修改答案 → 取消自动跳转
  - 修改后重新提交 → 显示新反馈

===============================================
技术实现细节：
===============================================

新增状态：
  - appState.canReedit: 是否可以修改答案
  - appState.autoNextTimer: 自动跳转计时器ID

新增函数：
  - prevQuestion(): 返回上一题
  - shareResult(): 分享成绩
  - showShareOptions(): 显示分享选项

修改函数：
  - submitAnswer(): 添加3秒自动跳转逻辑
  - selectOption(): 允许修改答案并取消自动跳转
  - showFeedback(): 显示自动跳转提示
  - showQuestion(): 更新按钮状态
  - stopAllTimers(): 清除自动跳转计时器

特别说明：

1. 所有用户答案仍会保存到：

   - 防止用户反复试错
   - 实现更真实的考试体验
