# 校服标准选择逻辑 - v3.1 更新说明

## 问题背景

在校服产品开发中，最常见的决策问题是：**如何根据纤维成分选择合适的标准执行？**

这个问题的出现原因：
- GB/T 31888-2015（2015年颁布的通用标准）规定：**夏季服装棉含量必须 ≥35%**
- 全涤纶（100% polyester）产品无法满足此要求
- 因此全涤纶产品需要改用 GB/T 23328（机织）或 GB/T 22854（针织）

## 新增题目（v3.1）

### ID 158：全涤纶产品的标准选择
**题目：** 某服装厂计划生产100%涤纶的夏季校服裤子，应该执行以下哪个标准？

**选项：**
- A. GB/T 31888-2015（中小学生校服）- ❌ 不合格（棉含量要求）
- B. GB/T 23328-2009（机织学生服装）- ✓ 正确
- C. 两个标准都可以执行 - ❌ 错误
- D. 无需执行任何标准 - ❌ 错误

**解释：** 
GB/T 31888-2015规定，夏季服装的棉含量必须≥35%。全涤纶产品无法满足此要求，应改为执行GB/T 23328-2009（机织学生服装标准）。这是行业内常见的做法：不同纤维成分的产品需要选择合适的标准执行。

---

### ID 159：GB/T 31888 vs GB/T 23328 的关键区别
**题目：** GB/T 31888-2015与GB/T 23328-2009在标准选择上有什么关键区别？

**选项：**
- A. GB/T 31888适用于所有校服，GB/T 23328已过时 - ❌ 错误
- B. GB/T 31888要求夏季棉含量≥35%；全涤产品应执行GB/T 23328 - ✓ 正确
- C. GB/T 31888仅适用于棉织面料 - ❌ 不准确
- D. 两个标准的技术要求完全相同 - ❌ 错误

**解释：** 
GB/T 31888-2015（2015年颁布）是通用校服标准，但规定了夏季服装棉含量≥35%的要求。GB/T 23328-2009是机织校服的基础标准。对于无法满足GB/T 31888棉含量要求的全涤纶产品，企业应执行GB/T 23328标准。这种标准选择在工艺师、设计师和品质部门的决策中非常重要。

---

### ID 160：工艺部门的标准选择决策
**题目：** 校服工艺部门收到一份全涤纶面料的夏季校服生产计划，应该如何确定执行的标准？

**选项：**
- A. 仍按GB/T 31888执行，不考虑棉含量要求 - ❌ 错误
- B. 与设计部门沟通，改变纤维成分以满足GB/T 31888的棉含量要求 - ✓ 正确
- C. 将标准改为GB/T 23328或GB/T 22854（视织物类型而定） - ✓ 也正确
- D. 不需要改变任何内容，任何标准都可以 - ❌ 错误

**解释：** 
工艺师的职责是确保选定的面料能够符合适用的标准。对于全涤纶面料，由于无法满足GB/T 31888-2015的夏季棉含量≥35%要求，应与设计部门沟通改用GB/T 23328（机织）或GB/T 22854（针织）。这是产品开发阶段必须解决的关键问题，避免后续生产中出现不合格。

## 适用岗位

这三道题特别重要的岗位：
- 👔 **设计师** - 需要了解纤维成分对标准适用性的影响
- 🧵 **工艺师** - 必须掌握：接到全涤产品时如何处理
- 📐 **版师** - 理解纤维成分与尺寸稳定性的关系
- ✅ **品质部** - 审核产品应执行的标准是否正确
- 📋 **项目经理** - 在产品立项时做出正确的标准选择

## 标准选择流程图

```
产品开发立项
     ↓
选择纤维成分
     ↓
   [决策点]
     ├─ 棉含量≥35%（棉 or 棉涤混纺）
     │    ↓
     │    执行 GB/T 31888-2015 ✓
     │    （通用校服标准）
     │
     └─ 棉含量<35% 或 100%涤纶
          ↓
          [织物类型]
          ├─ 机织面料 → 执行 GB/T 23328-2009 ✓
          └─ 针织面料 → 执行 GB/T 22854-2009 ✓
```

## 关键数据

| 指标 | GB/T 31888 | GB/T 23328 | GB/T 22854 |
|------|-----------|-----------|-----------|
| 适用范围 | 通用校服 | 机织校服 | 针织校服 |
| 发布年份 | 2015 | 2009 | 2009 |
| 夏季棉含量要求 | **≥35%** | 无强制 | 无强制 |
| 全涤纶产品 | ❌ 不合格 | ✓ 可用 | ✓ 可用 |
| 色牢度(耐皂洗) | ≥3级 | ≥3级 | ≥3级 |

## 实际应用案例

**案例1：棉涤混纺产品**
- 面料：65%棉 + 35%涤纶
- 棉含量：65% ≥ 35% ✓
- 选择标准：GB/T 31888-2015

**案例2：全涤纶机织产品**
- 面料：100%涤纶（机织）
- 棉含量：0% < 35% ✗
- 选择标准：GB/T 23328-2009

**案例3：全涤纶针织产品**
- 面料：100%涤纶（针织）
- 棉含量：0% < 35% ✗
- 选择标准：GB/T 22854-2009
