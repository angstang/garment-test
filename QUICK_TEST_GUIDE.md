## 🐛 故障排除

### 问题1: 看不到新功能（还是24道题）

**原因**: 浏览器缓存了旧版本

**解决方案**:
```
1. 按 Ctrl+Shift+R (Windows/Linux) 或 Cmd+Shift+R (Mac)
2. 或完全关闭浏览器重新打开
3. 或在隐身模式下测试
```

### 问题2: 多选题显示错误

**检查**:
1. 打开浏览器开发者工具 (F12)
2. 点击 Console 标签
3. 查看是否有红色错误信息
4. 向下滚动查看完整错误

**报告错误时提供**:
- 错误信息的完整文本
- 题号
- 使用的浏览器和版本

### 问题3: 问答题保存失败

**检查**:
1. 确保输入≥50个字
2. 查看控制台是否有错误
3. 尝试不同的浏览器
